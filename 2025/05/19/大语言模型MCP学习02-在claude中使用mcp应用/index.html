1.8 MCP应用工作流程

  1. MCP客户端将请求发送给Claude LLM
  2. Claude LLM分析可用的工具、决定使用哪个（些）工具
  3. MCP客户端通过MCP服务端执行LLM决定使用的那些工具
  4. 工具执行后的结果被返回给Claude LLM
  5. Claude LLM根据历史会话消息与工具执行后返回的结果，生成结果并组织成流畅可读的自然语言形式的回复
  6. 这些回复最终被展示在用户（你）的面前
